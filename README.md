# Gestion d'Entreprise

Application web de gestion d'entreprise développée en PHP, MySQL, HTML, CSS et JavaScript.

## Fonctionnalités

* Gestion des employés
* Gestion des clients
* Gestion des produits
* Gestion des ventes
* Gestion des factures
* Tableau de bord administrateur
* Tableau de bord employé
* Tableau de bord client
* Authentification et gestion des rôles
* Profil utilisateur
* Statistiques

## Technologies utilisées

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* WAMP Server

## Installation

1. Cloner le dépôt :

git clone https://github.com/andrechebouetoussa-cyber/gestion-entreprise.git

2. Copier le projet dans :

C:\wamp\www\gestion-entreprise

3. Créer la base de données MySQL.

4. Importer le fichier SQL du projet.

5. Configurer la connexion dans le fichier :

connexion.php

6. Lancer WAMP Server.

7. Accéder à l'application :

http://localhost/gestion-entreprise
